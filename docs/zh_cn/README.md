## 功能列表

- 启动/关闭游戏
- 收取荒原
- 每日心相（意志解析）
- 常规作战
- 活动刷取
- 自动深眠
- 自动醒梦
- 银行购物
- 领取奖励
- 雨中悬想：迷思海（"寻思"周扫荡）
- 切换账号
- 局外演绎：黄昏的音序
- 局外演绎：无声综合征
- 复刻活动推图
- 匣中交流赛（拓片杂技赛）
- 警铃鸣响时
- 翻斗棋
- 雨前漫游指南
- 自动推图（测试中）
