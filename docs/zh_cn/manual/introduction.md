# 功能介绍

## 启动游戏

启动游戏并等待进入主界面。

::: warning
暂不支持国际服PC端勾选运行该任务，如需该功能请到软件设置————启动设置修改相关配置
:::

## 收取荒原

收取荒原产出，包括以下选项：

- **好梦井**：收取好梦井产出
- **魔精收菜**：收取魔精生产物品
- **交付订单**：完成订单交付

## 每日心相（意志解析）

刷取意志解析每日免费次数，包括以下选项：

- **双重解析加成作战**：如有双重解析次数的情况下，使用体力进行作战
- **吃糖**：自动使用心相糖果

## 常规作战

执行常规关卡作战，包括以下选项：

- **自定义作战关卡**：手动指定作战关卡
  - 关闭时可选择关卡类型（主线/资源/洞悉）
  - 开启时可输入具体主线关卡编号
- **吃糖**：自动使用糖果
- **自定义作战次数**：指定作战次数，关闭时会清空体力
- **掉落统计上报**：上报关卡掉落数据

::: tip
12章更新了新材料，欢迎刷新关卡助力一图流更新！
:::

## 活动刷取

刷取活动关卡，包括以下选项：

- **复刻模式**：选择是否为复刻活动
- **活动难度**：选择活动关卡难度
- **吃糖**：自动使用糖果
- **自定义作战次数**：指定作战次数

::: warning
主线版本时关闭复刻模式会直接跳过当前任务，如有刷取主线关卡需求请使用常规作战功能
:::

## 自动深眠

自动完成深眠域战斗，包括以下选项：

- **上半编队**：选择上半区域使用的编队
- **下半编队**：选择下半区域使用的编队
- **仅领取每周浅眠域奖励**：只领取奖励不进行战斗

::: note

- 注意上下半编队不要重复
- 支持旧版和改版编队
- 支持编队自定义命名
- 未来可支持编队超出前四个
:::

## 自动醒梦

自动完成醒梦域战斗，包括以下选项：

- **醒梦上半编队**：选择上半区域使用的编队
- **醒梦下半编队**：选择下半区域使用的编队

::: note

- 注意上下半编队不要重复
- 支持旧版和改版编队
- 支持编队自定义命名
- 未来可支持编队超出前四个
:::

## 银行购物

自动在银行购买指定物品，包括以下选项：

- **单账号模式**：单账号模式下会跳过当前周期执行过的任务
- **柜台特奉**：购买柜台特奉商品
- **低阶柜台**：购买低阶柜台商品
- **心相观测**：购买心相观测
- **梦境记述**：购买梦境记述

## 领取奖励

领取各类奖励，包括以下选项：

- **领取邮件奖励**：收取邮件中的奖励
- **领取任务奖励**：领取日常、周常任务奖励
- **领取吼吼点唱机**：领取吼吼点唱机奖励

## 雨中悬想：迷思海

完成"寻思"每周扫荡任务。

- **单账号模式**：单账号模式下会跳过当前周期执行过的任务

::: note
本功能仅完成"寻思"每周扫荡
:::

## 切换账号

切换至最后一个账号（超过3个的账号会被忽略）。

::: note

- 切换后可继续添加任务，以便多账号多配置运行任务
- 目前该功能仅支持官服
- 建议将模拟器分辨率调至 1280×720

:::

## 关闭游戏

关闭游戏进程。

::: warning
暂不支持国际服PC端勾选运行该任务，如需该功能请到软件设置————启动设置修改相关配置
:::

## 局外演绎：黄昏的音序

肉鸽模式刷取信用功能，包括以下选项：

- **触媒选择**：选择使用的触媒（仅在非速刷模式生效）
- **速刷模式**：开启后仅使用术式神灯刷取第一层
- **演绎难度**：选择演绎难度

### 速刷模式

::: warning

1. 使用速刷功能前请先将 `刻画生长-指引之歌` `刻画生长-建筑章程` 点满
2. 请尽量调高难度等级（11难度达到最高信用效率200%）、点足科技树中信用效率加成，以防止分数不够 2k5 导致刷取信物失败

:::

### 非速刷模式

::: tip

1. 使用前可在箱子里置顶(标记)四个角色，作为出战角色
2. 队伍配置推荐携带一个奶
3. 使用前请保证最近一次战斗处于自动战斗状态，M9A不能检查是否处于自动战斗

:::

::: warning
M9A不对仅在初次玩肉鸽出现的剧情、引导等做额外处理，使用前需先自行通关一遍
:::

::: note
由于本功能实现较为复杂，容易出现奇怪的问题。如果有条件请开启模拟器的录屏，发现问题时尽量保留足够的信息（`debug` 以及 `logs` 目录下所有内容），以便开发者快速定位问题并解决。[提交问题点击此处](https://github.com/MAA1999/M9A/issues/new/choose)
:::

## 局外演绎：无声综合征

无声综合征肉鸽模式，包括以下选项：

- **战斗编队**：选择使用的战斗编队
- **携带好旅伴**：选择是否携带好旅伴
- **难度选择**：选择游戏难度
- **配器选择**：选择使用的配器

::: warning

1. M9A不对仅在初次玩肉鸽时出现的玩法说明页面做处理，请在使用前至少手动打到进入第二层
2. M9A不能检查是否处于自动战斗状态，使用前请保证最近一次战斗处于自动状态

:::

::: tip
经实验，最好选择难度2或者难度3进行任务，难度4容易战斗暴毙
:::

## 独立运行任务

以下任务需要在指定界面**单独运行**，不可与其他任务组合：

### 复刻活动推图

根据任务列表完成往期复刻活动推图，包括以下选项：

- **吃糖**：自动使用糖果

::: note
本任务根据任务列表完成任务，仅支持已打过的往期活动推图
:::

### 匣中交流赛（拓片杂技赛）

完成角斗场第一层扫荡。

::: warning
请手动打开角斗场页面后再运行此任务
:::

### 警铃鸣响时（需高练度）

完成警铃鸣响时扫荡，包括以下选项：

- **扫荡难度**：选择扫荡难度

::: warning
本功能不换角色，debuff吃满，故只适合高练队伍挂机
:::

::: tip

- 推荐配队：三巨头（薪电启）+图图石子、追击队
- 使用前请手动打开警铃鸣响时主页面（中间有雨迹追查和雨痕处置字样）
- **速刷选项**：直接打boss关，没有五号位buff，适合大成三巨头+双奶妈挂机通关
- 非速刷则会选择拉满五号位buff后再打boss，提高通关稳定性
- 测试用队伍：诺谛卡 + 圈 + 蘑菇 + 塞梅 + 图图（注意五号位放主C）
:::

### 翻斗棋速刷

在翻斗棋界面快速刷取。

::: warning

- 在翻斗棋界面启动
- 已支持国服3.2版本
- 使用前请先自行打两把使警戒值升到3级
:::

### 雨前漫游指南

完成雨前漫游指南小游戏，包括以下选项：

- **雨前漫游刷取模式**：根据教程进度选择对应模式

::: warning
该任务需单独运行，运行前请将游戏停在小游戏主页面
:::

::: note
该任务分为两种情况：

1. 未击败教程中第三天的boss（即点击"开始模拟"后不是难度选择界面）
2. 已击败教程中第三天的boss（即点击"开始模拟"后进入难度选择界面）

请根据个人情况选择对应选项
:::
